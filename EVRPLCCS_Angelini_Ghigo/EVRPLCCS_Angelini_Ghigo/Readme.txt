

Per eseguire il programma è necessario spostarsi col terminal nella cartella root ed eseguire "java -jar EVRPLCCS_Executable.jar <nomefile.txt> <secondi>".


I file di output saranno generati nella cartella output al cui interno è situata la cartella best dentro la quale sono presenti i file di output 
per tutte le 6 istanze provate sui 3 gap temporali ciascuno 5 volte.


Nel file output.csv sono presenti i risultati per ognuna delle 90 totali run e le medie sia per gap temporale che per istanza.
